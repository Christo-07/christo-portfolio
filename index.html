My Projects

ATM Simulator System – Java-Based Banking Application

Project Overview

The ATM Simulator System is a comprehensive Java-based desktop application that simulates the functionalities of a real-world Automated Teller Machine (ATM). Designed with a user-friendly graphical interface (Java Swing), this project demonstrates core banking operations, secure user authentication, and robust database integration using JDBC and MySQL.

Key Features

  • Multi-Step User Registration: Guided signup process collects personal, contact, and account details, generating a unique card number and PIN.
  • Secure Login: Users log in with card number and PIN, validated securely against the backend database.
  • Banking Operations:
    • Balance Enquiry: Instantly view current account balance.
    • Deposit & Withdrawal: Real-time balance updates with validation.
    • Mini Statement: View recent transaction history.
    • PIN Change: Change ATM PIN securely within the application.
  • Transaction Recording: All transactions are recorded in the database for accurate account management and traceability.
  • Database Integration: JDBC connects to MySQL for user data, authentication, and transaction records.
  • Intuitive GUI: Built with Java Swing, each operation is presented in a dedicated window for clarity and ease of use.

Technical Stack

  • Frontend: Java Swing (GUI)
  • Backend: Java (Core logic, JDBC)
  • Database: MySQL (User and transaction data storage)

Learning Highlights

  • Practical experience with Java GUI development and event-driven programming.
  • Implemented secure user authentication and sensitive data handling.
  • Designed and managed relational database schemas for banking operations.
  • Developed modular, maintainable code following OOP principles.

Typical User Flow

  1. User registers via the multi-step signup process.
  2. Logs in using card number and PIN.
  3. Accesses ATM operations: balance enquiry, deposit, withdrawal, mini statement, PIN change.
